Math Objects

Math objects Properties

  1. E - Euler's constant
  2. LN2 - Natural log of the value 2
  3. LN10 - Natural log of the value 10
  4. LOG2E - the base 2log of Euler constant
  5. LOG10E - the base 10 log of Euler constant (E)
  6. PI - returns the constant pi

Math Objects Methods

  1. abs(x) - returns the absolute value of x
  2. acos(x) - returns the arccosine of x, in radians
  3. asin(x) - returns the arcsine of x, in radians
  4. atan(x) - returns the arctangent of x as a numeric value between -PI/2 and PI/2 radians
  5. atan2(y, x) - returns the arctangent of the quotient of its arguments
  6. ceil(x) - returns x, rounded upwords to the nearest integer
  7. cos(x) - returns the cosine of x (x in radians)
  8. exp(x) - returns the value E**2
  9. floor(x) - returns x, rounded downwards to the nearest integer
  10. log(x) - returns the natural logarithm (base E) of x
  11. max(x, y, z, ..., n) - returns the number with the highest value
  12. min(x, y, z, ..., n) - returns the number with the lowest value
  13. pow(x, y) - returns the vaLUE OF X TO POWER Y
  14. random() - returns a random number between 0 and 1
  15. sin(x) - returns the sin of x (x in radians)
  16. sqrt(x) - returns the square root of x
  17. tan(x)- returns the tangen5t of an angle
